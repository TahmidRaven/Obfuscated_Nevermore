# 🐦 Raven Game

*A terminal-based arcade shooter written in pure C, featuring a flying raven, enemy birds, and devastating Caw-Blasts.*

## Overview

**Raven Game** is a compact, obfuscated terminal arcade game written in **C17**. Control a raven as it flies through the skies, dodging enemy birds and firing powerful **Caw-Blasts** to survive and rack up points.

## Features

- Animated raven sprite
- Projectile shooting ("Caw-Blast")
- Moving enemy birds
- Health system
- Score tracking
- Real-time keyboard input
- Terminal graphics using ASCII art
- Single-file implementation

## Controls

| Key | Action |
|-----|--------|
| `W` | Fly Up |
| `S` | Fly Down |
| `Space` | Fire Caw-Blast |
| `Q` | Quit Game |

## Compilation

```bash
gcc -std=c17 -O2 raven_game.c -o raven_game
```

## Running

```bash
./raven_game
```
